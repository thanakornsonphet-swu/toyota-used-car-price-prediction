# โครงการทำนายราคารถยนต์ Toyota มือสอง (Toyota Used Car Price Prediction)

   

## ภาพรวมโครงการ

**SWU Motors** ดีลเลอร์รถมือสองกำลังเผชิญปัญหาทางธุรกิจที่สำคัญ คือ **ยอดขายลดลงถึง 18%** สาเหตุหลักมาจากการตั้งราคาที่ผิดพลาดของพนักงานขายใหม่ (Junior Sales Staff)

โครงการ Data Science นี้จึงถูกจัดตั้งขึ้นเพื่อสร้าง **"เครื่องมือช่วยตั้งราคาอัจฉริยะ (AI Pricing Assistant)"** โดยใช้ Machine Learning เข้ามาช่วยประเมินราคากลาง เพื่อสร้างมาตรฐาน ลดความผิดพลาด และกู้คืนกำไรให้กับบริษัท

### วัตถุประสงค์

  * สร้างโมเดลทำนายราคาที่มีค่าความคลาดเคลื่อน (RMSE) **ต่ำกว่า £1,250**
  * ค้นหาปัจจัยสำคัญที่มีผลต่อราคา เพื่อเป็นแนวทางให้ฝ่ายจัดซื้อ

-----

## ข้อมูลที่ใช้ (Dataset)

ชุดข้อมูลการขายรถ Toyota มือสองจำนวน **6,738 รายการ**

  * **Source:** `toyota.csv`
  * **ตัวแปรตาม (Target):** `price` (ราคาขายหน่วยปอนด์)
  * **ตัวแปรต้น (Features):** รุ่นรถ (Model), ปีผลิต (Year), ระบบเกียร์ (Transmission), เลขไมล์ (Mileage), ชนิดเชื้อเพลิง (FuelType), ภาษี (Tax), อัตราสิ้นเปลือง (MPG), และ ขนาดเครื่องยนต์ (EngineSize)

-----

## กระบวนการทำงาน

เราดำเนินการตามขั้นตอนมาตรฐานของ Data Science Workflow:

1.  **Data Analytics (Excel & Tableau):**

      * พบว่าตลาดเป็นแบบ **"เบ้ขวา" (Right-Skewed)** รถส่วนใหญ่ราคาประหยัด แต่มีรถหรูจำนวนน้อยที่ดึงค่าเฉลี่ยขึ้น
      * ราคามีความผันผวนสูง (**High Volatility**) ทำให้การตั้งราคาด้วย "สัญชาตญาณ" ทำได้ยาก
      * **Insight:** "ขนาดเครื่องยนต์" เป็นปัจจัยที่แบ่งเกรดราคารถได้ชัดเจนที่สุด

2.  **Data Preprocessing (Python):**

      * แปลงข้อมูลหมวดหมู่ด้วย `OneHotEncoder`
      * ปรับสเกลข้อมูลตัวเลขด้วย `StandardScaler`
      * แบ่งข้อมูล Train 80% / Test 20%

3.  **Machine Learning Modeling:**

      * **Phase 1:** ทดสอบโมเดลเชิงเส้น (Linear Regression, Lasso) -\> *ผลลัพธ์ยังไม่ดีพอ*
      * **Phase 2:** เปลี่ยนมาใช้ **Random Forest Regressor** เพื่อจัดการความซับซ้อนของราคาแบบ Non-linear

-----

## ผลลัพธ์โมเดล (Model Performance)

เปรียบเทียบระหว่างโมเดลพื้นฐานและโมเดลที่นำเสนอ

| โมเดล (Model) | ค่าความคลาดเคลื่อน (RMSE) | ความแม่นยำ ($R^2$) | ผลลัพธ์ |
| :--- | :--- | :--- | :--- |
| **Linear Model (Lasso)** | £1,680.24 | 93.0% | ❌ **ไม่ผ่านเกณฑ์** (\> £1,250) |
| **Random Forest** | **£1,102.96** | **97.0%** | ✅ **ผ่านเกณฑ์** (\< £1,250) |

> **สรุป:** โมเดล Random Forest สามารถทำนายราคาได้แม่นยำสูง (Accuracy 97%) และลดความคลาดเคลื่อนลงได้ถึง **34%** เมื่อเทียบกับโมเดลเดิม ผ่านเกณฑ์ชี้วัดความสำเร็จที่ตั้งไว้

-----

## ข้อเสนอแนะทางธุรกิจ

จากการวิเคราะห์และผลลัพธ์โมเดล เรามีข้อเสนอแนะสำหรับ SWU Motors ดังนี้:

1.  **ใช้งานระบบ "Pricing Assistant":** นำโมเดลไปใช้หน้างานทันที เพื่อให้พนักงานขายมีราคากลางอ้างอิงที่แม่นยำ
2.  **คัดเลือกรถด้วย "Deal Analyzer":** ใช้โมเดลสแกนหารถที่ **"Undervalued"** (ราคาตลาดแพงกว่าราคาที่ประกาศขาย) เพื่อรับซื้อเข้าพอร์ตทำกำไร
3.  **เน้นรถเครื่องยนต์ใหญ่:** ทีมจัดซื้อควรเน้นหารถที่มีขนาดเครื่องยนต์ใหญ่ (SUV/Sport) เนื่องจากรักษามูลค่าได้ดีกว่าและมี Margin สูง
